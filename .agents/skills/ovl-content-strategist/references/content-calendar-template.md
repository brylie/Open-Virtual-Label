# Content Calendar Template

Monthly and quarterly calendar structures for independent artists.
Adapt these to the artist's actual platform mix and time budget from
`workspace/artists/<id>/content-strategy.md`.

---

## Monthly Calendar Structure

### Weekly rhythm

**Week 1 — Release or major update**
- If a release is live this week: release campaign content (see
  release-campaign-template.md)
- If no release: behind-the-scenes or work-in-progress from the most
  recent recording session

**Week 2 — Process and craft**
- Studio work documentation
- Creative process explanation
- Technical or production insights
- Honest reflections on the work

**Week 3 — Community and connection**
- Performance announcements (if upcoming)
- Response to comments or questions received
- Share or highlight others' work
- Community moments and gratitude

**Week 4 — Maintenance and buffer**
- Quick updates
- Archival content (older releases worth revisiting)
- Playlist or listening context
- Flexible week — absorbs unexpected content needs or rests if needed

**Throughout the month:**
- Stories: casual, low-effort, 2–4× per week on primary platform
- Secondary platforms: cross-post where it makes sense; minimal extra effort

## Content Type Reference

| Type | Description | Effort | Frequency |
| ---- | ----------- | ------ | --------- |
| Release announcement | Album/single is out — lead with the music | Medium | Per release |
| Track spotlight | Highlight one track from the catalog, with context | Low | 1–2× / month |
| Behind-the-scenes | Studio setup, process clips, work-in-progress | Very low | Capture during creation |
| Performance announcement | Venue, date, what to expect | Low | Per performance (1 week out) |
| Performance recap | Photos or reflection after the show | Low | Day after show |
| Process/craft | How something was made, a decision, a technique | Low–Medium | 1–2× / month |
| Community engagement | Responding, sharing, gratitude | Very low | Ongoing |
| Milestone | Streaming numbers, release anniversary, notable moment | Very low | When it occurs |
| Seasonal content | Tie to time of year — only when genuinely relevant | Low | Major seasons |

## Promotional vs. Engagement Balance

Read the target ratio from `content-strategy.md`. General guidance:

| Category | Typical range | Examples |
| -------- | ------------- | -------- |
| Promotional | 30–40% | Release announcements, pre-save links, streaming links, merch |
| Engagement | 60–70% | Behind-the-scenes, process, performance recaps, community |

Skewing too promotional erodes trust. Skewing too far toward engagement
can mean releases land without traction. Adjust each month based on
whether a release is active.

---

## Planned vs. Spontaneous Mix

- **Planned (~70%):** Release campaigns, performance announcements, scheduled
  batch content. Prevents gaps and reduces daily decision-making.
- **Spontaneous (~30%):** Real-time updates, unexpected moments, authentic
  reactions. Keeps the calendar from feeling manufactured.

Leave explicit buffer in weeks 4 of each month for spontaneous content.
