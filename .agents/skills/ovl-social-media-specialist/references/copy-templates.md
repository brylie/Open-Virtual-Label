# Copy Templates

Structural guides for common content types. These define what information
goes where — not the words. Fill in with the artist's specific facts and
voice from `workspace/artists/<id>/social-media.md`.

**Never copy these verbatim.** Replace every `[bracketed]` item with real,
specific content before presenting to the artist.

---

## Album / EP announcement

**Facts needed from artist:** title, release date or "out now", where to
listen, optional: brief description or creation context.

### Instagram

```
[Title] out now. [One sentence describing what it is.]

[Optional: One sentence about the creation context.]

Listen on [platform], [platform], [platform] (link in bio)

[CC licensing line from social-media.md]

[Hashtags]
```

### YouTube

```
TITLE:
[Album Title] — [Type, e.g. Full Album] | [Artist Name] (X/60)

DESCRIPTION:
[Album Title] is out now on [platforms].

[2–3 sentences describing the album and creation context. Pull from epk.md
or the artist's description — do not invent.]

Listen:
[Platform]: [link]
[Platform]: [link]

[CC licensing paragraph from social-media.md]

Subscribe for more [genre description].

TAGS:
[genre], [genre], [artist name], creative commons music, [year]
```

## Livestream announcement

**Facts needed:** platform, date, time (include timezone), what they're
working on or playing.

### Instagram story text (brief, for overlay)

```
Live on [Platform] [day] at [time] [timezone]
[What they're working on]
Link in bio
```

### Instagram post (if a feed post is appropriate)

```
Live on [YouTube / Platform] [day] at [time].

[What they'll be working on — specific. Not "making music" but "finishing
the last two tracks for [Album]."]

[Casual close — "come hang out" or equivalent in the artist's voice]

[Hashtag for livestream + genre]
```

### YouTube community post

```
Live in [X hours / at time].

[What they're working on — specific.]

[Brief, low-key CTA. Ask a question or invite lurkers.]

[Link]
```

the artist's music. Coordinate with `ovl-licensing-outreach` for the

link to project if shareable.

### Instagram

```

to the artist's audience.]
```

## CC licensing language reference

Pull the exact wording from `workspace/artists/<id>/social-media.md →
cc_framing`. Common structures:

```
Creative Commons licensed — free to use in your projects with attribution.
CC BY 4.0 — free for podcasts, videos, and games.
All music Creative Commons — use it, just credit me.
```

Do not invent new CC framing; use what the artist has approved.
