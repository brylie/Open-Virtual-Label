## Notes on completing the template

- **Executive summary** is written last — after all sections are filled in
- **3-month average** requires data from at least 3 snapshot files in
  `workspace/artists/<id>/metrics/`; use what's available and note when
  fewer months are available
- **Forecasts** use the formulas in
  [metrics-calculations.md](metrics-calculations.md) — show the calculation
  briefly so the artist can verify
- **Celebrations** should include at least one item; find something genuine
  even in a slow month
