# Artist Identity

Narrative context for the `ovl-artist-manager` skill. Structured data
(platforms, license, distributor, PRO) lives in `artist.json` — read that
first. This file captures what the JSON schema cannot: philosophy, constraints,
persona detail, and long-term vision.

Copy this file to `workspace/artists/<artist-id>/artist-identity.md` and
fill in the values.
