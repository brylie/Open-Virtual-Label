# Financial Report Templates

Use these templates to produce consistent, interpretive financial reports.
For the underlying formulas, see [calculation-methods.md](calculation-methods.md).

**Tone guidelines:** Lead with what's working. Be honest about challenges but
frame them constructively. Quantify wherever possible. End with forward-looking
clarity. Celebrate progress explicitly, even if small.

---
